## 🏁 Getting Started <a name = "getting_started"></a>
To build this web application I used the following:
<ul>
<li>ReactJS</li>
<li>Axios</li>
<li>Redux-Thunk</li>
<li>React Router</li>
<li>Styled Components</li>
</ul>

~To open the Development Server
```
npm start
```
~To build the project from Deployment:
```
npm run build
```
